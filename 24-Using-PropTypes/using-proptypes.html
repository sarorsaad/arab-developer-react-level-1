Lesson Twenty-Four: Using PropTypes

الدرس الرابع والعشرون: استخدام PropTypes

بسم الله الرحمن الرحيم

في هذا الدرس، سنتعلم كيفية استخدام PropTypes للتحقق من أنواع البيانات في مكونات React. يساعدنا PropTypes في التحقق من صحة أنواع البيانات التي يتم تمريرها إلى مكوناتنا، مما يجعل تطبيقاتنا أكثر موثوقية وأسهل في التصحيح.

In this lesson, we'll learn how to use PropTypes for type checking in React components. PropTypes help us validate the data types of props passed to our components, making our applications more reliable and easier to debug.

تثبيت PropTypes | Installing PropTypes

أولاً، نحتاج إلى تثبيت حزمة PropTypes:

First, we need to install the PropTypes package:

npm install prop-types

الاستخدام الأساسي | Basic Usage

هذا مثال أساسي على استخدام PropTypes:

Here's a basic example of using PropTypes:

import React from 'react';
import PropTypes from 'prop-types';

class UserProfile extends React.Component {
    render() {
        return (
            <div>
                <h2>{this.props.name}</h2>
                <p>Age: {this.props.age}</p>
            </div>
        );
    }
}

UserProfile.propTypes = {
    name: PropTypes.string.isRequired,
    age: PropTypes.number
};

أنواع PropTypes المتاحة | Available PropTypes

يتضمن React PropTypes العديد من المدققات:

React PropTypes includes several validators:

  • PropTypes.string | للنصوص
  • PropTypes.number | للأرقام
  • PropTypes.bool | للقيم المنطقية
  • PropTypes.array | للمصفوفات
  • PropTypes.object | للكائنات
  • PropTypes.func | للدوال

الخصائص المطلوبة | Required Props

يمكنك تحديد خاصية كمطلوبة بإضافة isRequired:

You can mark a prop as required by adding .isRequired:

MyComponent.propTypes = {
    requiredString: PropTypes.string.isRequired,
    requiredNumber: PropTypes.number.isRequired
};

PropTypes المركبة | Complex PropTypes

يدعم PropTypes أيضًا الأنواع الأكثر تعقيدًا:

PropTypes also supports more complex types:

MyComponent.propTypes = {
    // Array of numbers | مصفوفة من الأرقام
    numbers: PropTypes.arrayOf(PropTypes.number),

    // Object with specific shape | كائن بشكل محدد
    user: PropTypes.shape({
        name: PropTypes.string,
        age: PropTypes.number
    }),

    // One of specific values | واحد من قيم محددة
    status: PropTypes.oneOf(['active', 'inactive'])
};

رسائل الخطأ | Error Messages

عندما يفشل التحقق من نوع الخاصية، سيظهر React تحذيرًا في وحدة التحكم أثناء التطوير:

When a prop type validation fails, React will show a warning in the console during development:

Warning: Failed prop type: Invalid prop `age` of type `string` 
supplied to `UserProfile`, expected `number`.

أفضل الممارسات | Best Practices

  • دائمًا حدد PropTypes لمكوناتك | Always define PropTypes for your components
  • استخدم isRequired للخصائص الإلزامية | Use .isRequired for mandatory props
  • وثّق أنواع الخصائص المعقدة | Document complex prop types
  • قم بإزالة PropTypes في الإنتاج | Remove PropTypes in production

الملخص | Summary

PropTypes أداة قوية للتحقق من الأنواع في React. تساعد في اكتشاف الأخطاء مبكرًا وتجعل الكود الخاص بك أكثر قابلية للصيانة.

PropTypes is a powerful tool for type checking in React. It helps catch bugs early and makes your code more maintainable.
