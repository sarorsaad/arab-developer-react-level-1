Lesson Fourteen: What Are Components in React

الدرس الرابع عشر: ما هي المكونات في React

المكونات هي اللبنات الأساسية لتطبيقات React. تسمح لك بتقسيم واجهة المستخدم إلى أجزاء مستقلة وقابلة لإعادة الاستخدام.
Components are the building blocks of React applications. They allow you to split the UI into independent, reusable pieces.

ما هي المكونات؟ | What are Components?

المكون هو دالة JavaScript أو class تقبل مدخلات (props) وتعيد عناصر React تصف ما يجب أن يظهر على الشاشة.
A component is a JavaScript function or class that accepts inputs (props) and returns React elements describing what should appear on the screen.

لماذا نستخدم المكونات؟ | Why Use Components?

1. إعادة الاستخدام | Reusability

يمكن إعادة استخدام المكونات في جميع أنحاء تطبيقك، مما يقلل من تكرار الكود.
Components can be reused throughout your application, reducing code duplication.

2. قابلية الصيانة | Maintainability

تجعل المكونات الكود الخاص بك أسهل في الصيانة حيث يكون كل مكون مسؤولاً عن منطقه الخاص.
Components make your code easier to maintain as each component is responsible for its own logic.

3. سهولة القراءة | Readability

تقسيم واجهة المستخدم إلى مكونات يجعل الكود أكثر قابلية للقراءة وأسهل للفهم.
Breaking down your UI into components makes the code more readable and easier to understand.

أنواع المكونات | Types of Components

1. المكونات الوظيفية | Function Components


function Welcome(props) {
    return 
Hello, {props.name}
;
}

2. مكونات الفئة | Class Components


class Welcome extends React.Component {
    render() {
        return 
Hello, {this.props.name}
;
    }
}

أفضل الممارسات | Best Practices

  • المسؤولية الواحدة: يجب أن يقوم كل مكون بشيء واحد بشكل جيد
    Single Responsibility: Each component should do one thing well
  • لا تكرر نفسك: إذا كررت نفس الكود، فكر في إنشاء مكون
    DRY: If you repeat code, consider creating a component
  • أسماء ذات معنى: استخدم أسماء واضحة ووصفية لمكوناتك
    Meaningful Names: Use clear, descriptive names for your components
