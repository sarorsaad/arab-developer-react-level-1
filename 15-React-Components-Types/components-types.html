Lesson Fifteen: React Components Types

الدرس الخامس عشر: أنواع المكونات في React

في React، هناك نوعان رئيسيان من المكونات: Class Components و Function Components.
In React, there are two main types of components: Class Components and Function Components.

Class Components | مكونات الفئة

Class Components هي طريقة قديمة (legacy) لكتابة المكونات في React. تستخدم كلمة class وتمتد من React.Component.
Class Components are a legacy way of writing React components. They use the class keyword and extend React.Component.

كتابة Class Component | Writing a Class Component

لكتابة Class Component، نحتاج إلى:
To write a Class Component, we need:

  • كلمة class
    The class keyword
  • extends React.Component
    Extending React.Component
  • طريقة render()
    The render() method 

import React from 'react';

class HelloWorld extends React.Component {
    render() {
        return (
            
Hello, World!

        );
    }
}

export default HelloWorld;

طريقة Render | The Render Method

طريقة render() هي طريقة إجبارية في Class Components. تحدد ما سيظهر في المتصفح.
The render() method is mandatory in Class Components. It defines what will be displayed in the browser.

  • يجب أن تعيد JSX
    Must return JSX
  • تُستدعى تلقائياً عند تحديث المكون
    Called automatically when component updates
  • لا يمكن أن تكون async
    Cannot be async

ملاحظة عن Class Components | Note About Class Components

رغم أن Class Components لا تزال مدعومة، إلا أن React تُوصي باستخدام Function Components مع Hooks.
Although Class Components are still supported, React recommends using Function Components with Hooks.
