React Practical Guide: Creating Elements & Behind-the-Scenes Process

الدليل العملي لـ React: إنشاء العناصر والعمليات خلف الكواليس

في هذا الدرس، سنتعلم كيفية إنشاء عناصر React بشكل عملي، وفهم ما يحدث خلف الكواليس عند كتابة كود React.

إنشاء العناصر (Creating Elements)

// Creating an H1 element
const heading = React.createElement(
    'h1',
    null,
    'Hello and Welcome'
);

// Creating a paragraph element
const paragraph = React.createElement(
    'p',
    null,
    'محمد مصطفى'
);

// Creating a parent div
const parentDiv = React.createElement(
    'div',
    null,
    [heading, paragraph]
);

شرح الكود:

  • إنشاء عنصر H1 باستخدام React.createElement
  • إنشاء عنصر paragraph للنص العربي
  • إنشاء div رئيسي يحتوي على العناصر السابقة

ما يحدث خلف الكواليس (Behind the Scenes)

عند إنشاء عناصر React، تتم العملية على عدة مراحل:

  1. تحويل كل عنصر إلى كائن JavaScript (JavaScript Object)
  2. إنشاء عقد Fiber (Fiber Nodes)
  3. ربط العقد في شجرة Fiber (Fiber Tree)
  4. إنشاء جذر React (React Root)

مثال عملي (Practical Example)

// Create Root
const root = ReactDOM.createRoot(
    document.getElementById('root')
);

// Render the content
root.render(parentDiv);

شجرة Fiber (Fiber Tree)

شجرة Fiber هي تمثيل داخلي للـ Virtual DOM، حيث:

  • كل عنصر يمثل عقدة (Node)
  • العقد مرتبطة ببعضها البعض
  • تحتوي على معلومات عن العناصر وعلاقاتها
