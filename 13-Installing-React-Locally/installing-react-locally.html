Lesson Thirteen: Installing React Locally

الدرس الثالث عشر: تثبيت React محلياً

في هذا الدرس سنتعلم كيفية تثبيت React محلياً باستخدام طريقتين: Create React App و Vite.
In this lesson, we'll learn how to install React locally using two methods: Create React App and Vite.

لماذا التثبيت المحلي؟ | Why Local Installation?

مشاكل استخدام CDN Links:
Problems with using CDN Links:

  • بطء في الأداء
    Performance issues
  • عدم توفر Hot Module Replacement (HMR)
    No Hot Module Replacement (HMR)
  • بطء في ترجمة JSX
    Slower JSX compilation
  • صعوبة إدارة المكتبات الإضافية
    Difficult dependency management

Create React App (CRA) | إنشاء تطبيق React باستخدام CRA

# Using npm
npm install -g create-react-app
create-react-app my-app

# Using npx (recommended)
npx create-react-app my-app

# Starting the development server
cd my-app
npm start

ملاحظة: CRA يستغرق وقتاً أطول في التثبيت (3-5 دقائق)
Note: CRA takes longer to install (3-5 minutes)

Vite Method | إنشاء تطبيق React باستخدام Vite

// تثبيت React محلياً
npm create vite@latest my-react-app -- --template react

// الدخول إلى مجلد المشروع
cd my-react-app

// تثبيت المكتبات المطلوبة
npm install

// تشغيل المشروع
npm run dev

ملاحظة: Vite أسرع بكثير في التثبيت والتشغيل
Note: Vite is much faster in installation and execution

المقارنة بين الطريقتين | Comparison
