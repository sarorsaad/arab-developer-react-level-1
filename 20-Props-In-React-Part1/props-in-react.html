Lesson Twenty: Props in React Part 1

الدرس العشرون: Props في React - الجزء الأول

بسم الله الرحمن الرحيم

في الدرس السابق، قمنا بإنشاء مشروع بسيط يحتوي على ثلاثة مكونات متشابهة. اليوم سنتعلم كيف نجعل هذه المكونات أكثر كفاءة باستخدام Props.

In our previous lesson, we created a simple project with three similar components. Today, we'll learn how to make these components more efficient using Props.

مفهوم Props | Understanding Props

Props (اختصار لـ Properties) هي الطريقة التي نمرر بها البيانات من المكون الأب إلى المكون الابن في React. تعمل Props مثل الـ parameters في الدوال العادية.

Props (short for Properties) is how we pass data from parent components to child components in React. Props work similarly to parameters in regular functions.

المكونات الثابتة مقابل الديناميكية | Static vs Dynamic Components

لنقارن بين المكونات الثابتة والديناميكية:

Let's compare static and dynamic components:

// المكون الثابت (الطريقة غير المثالية)
// Static Component (Not Ideal)
class Project extends React.Component {
render() {
return (
<div className="project">
<h3>Admin Dashboard</h3>
<p>A powerful admin dashboard</p>
</div>
);
}
}
// المكون الديناميكي باستخدام Props (الطريقة المثالية)
// Dynamic Component using Props (Ideal)
class Project extends React.Component {
render() {
return (
<div className="project">
<h3>{this.props.title}</h3>
<p>{this.props.description}</p>
</div>
);
}
}

استخدام Props | Using Props

طريقة استخدام المكون الديناميكي:

How to use the dynamic component:

// استخدام المكون الديناميكي
// Using the Dynamic Component
<Project
title="Admin Dashboard"
description="A powerful admin dashboard"
/>

فوائد Props | Key Benefits of Props

  • Code Reusability (إعادة استخدام الكود):

    نستطيع استخدام نفس المكون مع بيانات مختلفة.

    We can use the same component with different data.

  • Dynamic Content (المحتوى الديناميكي):

    يمكننا تغيير محتوى المكون بسهولة عن طريق تغيير الـ props.

    We can easily change component content by changing props.

  • Maintainability (سهولة الصيانة):

    الكود يصبح أسهل في الصيانة والتعديل.

    Code becomes easier to maintain and modify.

ملخص الدرس | Summary

Props هي الطريقة الأساسية لجعل مكونات React ديناميكية وقابلة لإعادة الاستخدام. من خلال Props، يمكننا تمرير البيانات من المكون الأب إلى المكون الابن، مما يجعل تطبيقاتنا أكثر مرونة وكفاءة.

Props are the fundamental way to make React components dynamic and reusable. Through Props, we can pass data from parent to child components, making our applications more flexible and efficient.
