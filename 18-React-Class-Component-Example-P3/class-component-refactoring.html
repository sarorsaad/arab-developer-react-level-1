React Class Component Example P3: Refactoring Code

إعادة هيكلة الكود باستخدام Class Component

بسم الله الرحمن الرحيم

في هذا الدرس سنقوم بإعادة هيكلة الكود (Refactoring) باستخدام Class Components وتنظيمه بشكل أفضل.

In this lesson, we'll refactor our code using Class Components and organize it in a better way.

App Component إنشاء | Creating App Component

سنقوم بإنشاء مكون رئيسي App يحتوي على كل المكونات الأخرى:

We'll create a main App component that contains all other components:

// App.jsx
import React from 'react';

class App extends React.Component {
    render() {
        return (
            <div className="parent">
                <Header />
                <MainContent />
                <Sidebar />
                <Footer />
            </div>
        );
    }
}

export default App;

إنشاء مجلد Components | Creating Components Folder

نقوم بإنشاء مجلد components لتنظيم المكونات:

Create a components folder to organize our components:

src/
  ├── components/
  │   ├── Header.jsx
  │   ├── MainContent.jsx
  │   ├── Sidebar.jsx
  │   └── Footer.jsx
  ├── App.jsx
  └── main.jsx

Header Component

نقوم بإنشاء مكون Header في ملف منفصل:

Create Header component in a separate file:

// components/Header.jsx
import React from 'react';

class Header extends React.Component {
    render() {
        return (
            <header>
                <h1>Welcome to my website</h1>
            </header>
        );
    }
}

export default Header;

Main Content Component

نقوم بإنشاء مكون MainContent في ملف منفصل:

Create MainContent component in a separate file:

// components/MainContent.jsx
import React from 'react';

class MainContent extends React.Component {
    render() {
        return (
            <main className="main-content">
                <h2>Main Content</h2>
                <p>This is the main content area.</p>
            </main>
        );
    }
}

export default MainContent;

Sidebar Component

نقوم بإنشاء مكون Sidebar في ملف منفصل:

Create Sidebar component in a separate file:

// components/Sidebar.jsx
import React from 'react';

class Sidebar extends React.Component {
    render() {
        return (
            <aside className="sidebar">
                <h3>Navigation</h3>
                <ul>
                    <li><a href="#">Link 1</a></li>
                    <li><a href="#">Link 2</a></li>
                    <li><a href="#">Link 3</a></li>
                </ul>
            </aside>
        );
    }
}

export default Sidebar;

Footer Component

نقوم بإنشاء مكون Footer في ملف منفصل:

Create Footer component in a separate file:

// components/Footer.jsx
import React from 'react';

class Footer extends React.Component {
    render() {
        return (
            <footer>
                <p>© 2024 My Website. All rights reserved.</p>
            </footer>
        );
    }
}

export default Footer;

عملية الـ Rendering | Rendering Process

عملية الـ Rendering تتم كالتالي:

The Rendering process works as follows:

  1. يبدأ من main.jsx (نقطة البداية)
    Starts from main.jsx (entry point)
  2. يقوم بربط Virtual DOM مع Real DOM
    Links Virtual DOM with Real DOM
  3. يقوم بتنفيذ render() لكل مكون
    Executes render() for each component
  4. يقوم بإنشاء شجرة العناصر
    Creates the element tree
  5. يقوم بعرض النتيجة النهائية
    Displays the final result
