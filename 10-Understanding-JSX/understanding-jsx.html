Lesson Ten: Understanding JSX

الدرس العاشر: فهم JSX

في هذا الدرس سنتعلم عن JSX وكيف يتم تحويله إلى JavaScript باستخدام Babel. سنفهم لماذا نستخدم JSX وما هي مميزاته.
In this lesson, we'll learn about JSX and how it gets transformed into JavaScript using Babel. We'll understand why we use JSX and its benefits.

ما هو JSX؟ | What is JSX?

JSX هو امتداد لـ JavaScript يسمح لنا بكتابة HTML داخل ملفات JavaScript. المصطلح JSX يعني JavaScript XML.
JSX is a JavaScript extension that allows us to write HTML inside JavaScript files. The term JSX stands for JavaScript XML. 


// مثال بسيط على JSX
const element = (
    <div>
        <h1>مرحباً بالعالم</h1>
        <p>هذا مثال على JSX</p>
    </div>
);

تحويل JSX إلى JavaScript | JSX to JavaScript Transformation

المتصفح لا يفهم JSX مباشرة. يتم استخدام Babel لتحويل كود JSX إلى JavaScript عادي.
Browsers don't understand JSX directly. Babel is used to transform JSX code into regular JavaScript. 

// Before Babel (JSX)
const element = (
    <div>
        <h1>Hello</h1>
        <p>Welcome</p>
    </div>
);

// After Babel (React 17+)
import { jsx as _jsx, jsxs as _jsxs } from 'react/jsx-runtime';

const element = _jsxs("div", {
    children: [
        _jsx("h1", { children: "Hello" }),
        _jsx("p", { children: "Welcome" })
    ]
});

عملية التحويل في Babel | Babel Transformation Process

يمر كود JSX بعدة مراحل في Babel:
JSX code goes through several stages in Babel:

  1. تحليل الكود (Parsing)
    Code Parsing
  2. تحويل JSX إلى وظائف React (Transform)
    Transforming JSX to React Functions
  3. توليد الكود النهائي (Code Generation)
    Final Code Generation

مميزات JSX | Benefits of JSX

  • سهولة القراءة والكتابة
    Easy to read and write
  • يمكن استخدام JavaScript expressions داخل JSX
    Can use JavaScript expressions inside JSX
  • أداء أفضل في React 17+
    Better performance in React 17+
  • لا يحتاج إلى import React في كل ملف
    No need to import React in every file

مثال عملي | Practical Example

مثال على استخدام التعبيرات JavaScript داخل JSX:
Example of using JavaScript expressions inside JSX: 


// استخدام التعبيرات في JSX
const name = 'أحمد';
const element = (
    <div>
        <h1>مرحباً {name}</h1>
        <p>الوقت الحالي هو: {new Date().toLocaleTimeString()}</p>
    </div>
);
