Lesson Seventeen: React Class Component Part 2

الدرس السابع عشر: مكونات الفئة في React - الجزء الثاني

في هذا الدرس، سنقوم بإعادة هيكلة الموقع الذي أنشأناه في الدرس السابق باستخدام Class Components.

In this lesson, we'll refactor the website we created in the previous lesson using Class Components.

فصل المكونات | Component Separation

سنقوم بتقسيم الموقع إلى أربعة مكونات رئيسية:

We'll split the website into four main components:

  • Header Component (مكون الهيدر)
  • Main Content Component (مكون المحتوى الرئيسي)
  • Sidebar Component (مكون الشريط الجانبي)
  • Footer Component (مكون التذييل)

Header Component | مكون الهيدر

نقوم بإنشاء مكون الهيدر كـ Class Component:

Creating the Header as a Class Component:


// المكون الرئيسي
class Header extends React.Component {
    render() {
        return (
            <header>
                <h1>موقعي</h1>
                <nav>
                    <ul>
                        <li>الرئيسية</li>
                        <li>المنتجات</li>
                        <li>اتصل بنا</li>
                    </ul>
                </nav>
            </header>
        );
    }
}

Main Content Component | مكون المحتوى الرئيسي

نقوم بإنشاء مكون المحتوى الرئيسي:

Creating the Main Content component:


// مكون المحتوى الرئيسي
class MainContent extends React.Component {
    render() {
        return (
            <main>
                <h2>مرحباً بكم في موقعنا</h2>
                <p>نحن نقدم أفضل المنتجات والخدمات</p>
            </main>
        );
    }
}

Sidebar Component | مكون الشريط الجانبي

نقوم بإنشاء مكون الشريط الجانبي:

Creating the Sidebar component:

class Sidebar extends React.Component {
    render() {
        return (
            <aside className="sidebar">
                <h3>Navigation</h3>
                <ul>
                    <li><a href="#">Link 1</a></li>
                    <li><a href="#">Link 2</a></li>
                    <li><a href="#">Link 3</a></li>
                </ul>
            </aside>
        );
    }
}

Footer Component | مكون التذييل

نقوم بإنشاء مكون التذييل:

Creating the Footer component:


// مكون التذييل
class Footer extends React.Component {
    render() {
        return (
            <footer>
                جميع الحقوق محفوظة © 2025
            </footer>
        );
    }
}

الكود النهائي | Final Code

الآن نقوم بتجميع كل المكونات في الملف الرئيسي:

Now we combine all components in the main file:

import React from 'react';
import { createRoot } from 'react-dom/client';

// Import our components
import Header from './components/Header';
import MainContent from './components/MainContent';
import Sidebar from './components/Sidebar';
import Footer from './components/Footer';

createRoot(document.getElementById('root')).render(
    <div className="parent">
        <Header />
        <MainContent />
        <Sidebar />
        <Footer />
    </div>
);

التصحيح والتشغيل | Debugging and Running

لتشغيل التطبيق، نتبع الخطوات التالية:

To run the application, follow these steps:

  1. تأكد من وجودك في المجلد الصحيح
    Make sure you're in the correct directory
  2. قم بتشغيل الأمر: npm run dev
    Run the command: npm run dev
  3. تأكد من استيراد React في كل مكون
    Ensure React is imported in each component
