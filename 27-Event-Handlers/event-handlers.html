Lesson Twenty-Seven: Event Handlers

الدرس السابع والعشرون: معالجات الأحداث

بسم الله الرحمن الرحيم

في هذا الدرس، سنتعلم كيفية التعامل مع الأحداث (Events) في React، وهو موضوع أساسي للتفاعل مع المستخدم.

In this lesson, we'll learn how to handle events in React, which is a fundamental topic for user interaction.

صيغة كتابة الأحداث | Event Syntax


// تعريف الحدث باستخدام onClick
<button onClick={handleClick}>اضغط هنا</button>

// تعريف الدالة المعالجة للحدث
function handleClick() {
    alert("تم النقر على الزر");
}

كتابة معالجات الأحداث | Writing Event Handlers


class MyComponent extends React.Component {
    // تعريف دالة معالجة النقر
    handleClick = () => {
        alert("تم النقر!");
    }

    render() {
        return (
            // استخدام الدالة مع الحدث
            <button onClick={this.handleClick}>
                انقر هنا
            </button>
        );
    }
}

الدوال المباشرة مقابل المنفصلة | Inline vs Separated Functions


// الطريقة المباشرة
<button onClick={() => alert("مرحباً")}>
    اضغط هنا
</button>

// الطريقة المنفصلة (مفضلة)
handleClick = () => {
    alert("مرحباً");
}

<button onClick={this.handleClick}>
    اضغط هنا
</button>

الأخطاء الشائعة | Common Mistakes


// ❌ خطأ: استدعاء الدالة مباشرة
<button onClick={this.handleClick()}>

// ✅ صحيح: تمرير مرجع الدالة
<button onClick={this.handleClick}>

// ✅ صحيح أيضاً: استخدام arrow function
<button onClick={() => this.handleClick()}>

اتفاقيات التسمية | Naming Conventions

  • أسماء الأحداث تبدأ بـ on مثل onClick, onChange
  • أسماء معالجات الأحداث تبدأ بـ handle مثل handleClick, handleChange

ملخص الدرس | Lesson Summary

  • الأحداث في React تكتب بنمط camelCase
  • نمرر دوال كمعالجات للأحداث
  • يمكن كتابة الدوال مباشرة أو بشكل منفصل
  • لا نستدعي الدوال مباشرة في الأحداث
  • نتبع اتفاقيات التسمية المعيارية
