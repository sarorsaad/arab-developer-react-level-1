Lesson Eight: React Create Element

الدرس الثامن: إنشاء عنصر في React

Step-by-Step What Is Going Under The Hood | خطوة بخطوة: ما يحدث خلف الكواليس

1. React.createElement Process | عملية إنشاء العنصر

1

Creating Element | إنشاء العنصر


// إنشاء عنصر بسيط
const element = React.createElement(
    'p',
    null,
    'مرحباً بالعالم'
);

JavaScript Object Representation | التمثيل ككائن JavaScript


{
    type: 'p',
    props: null,
    children: ['مرحباً بالعالم']
}

2. Creating Root | إنشاء الجذر

2

ReactDOM.createRoot | إنشاء نقطة الاتصال


// إنشاء عنصر مع خصائص
const element = React.createElement(
    'button',
    {
        className: 'btn',
        onClick: () => alert('تم النقر!')
    },
    'انقر هنا'
);

Fiber Root Creation | إنشاء جذر Fiber

  • Creates connection point between Virtual DOM and Real DOM
  • ينشئ نقطة اتصال بين DOM الافتراضي و DOM الحقيقي

3. Render Process | عملية العرض

3

root.render() | عملية العرض


// إنشاء عنصر مع عناصر متداخلة
const element = React.createElement(
    'div',
    { className: 'container' },
    React.createElement('h1', null, 'العنوان'),
    React.createElement(
        'p',
        { className: 'content' },
        'هذا هو المحتوى'
    ),
    React.createElement(
        'button',
        { onClick: () => alert('مرحباً') },
        'انقر هنا'
    )
);

Render Phases | مراحل العرض:

Reconciliation Phase | مرحلة المطابقة
  • Compares Virtual DOMs | يقارن بين DOM الافتراضي القديم والجديد
  • Identifies changes | يحدد التغييرات
Commit Phase | مرحلة التنفيذ
  • Updates Real DOM | يحدث DOM الحقيقي
  • Applies changes | يطبق التغييرات

4. Live Example | مثال حي

Code | الكود


// مثال متقدم: نموذج تسجيل
const element = React.createElement(
    'form',
    { onSubmit: (e) => e.preventDefault() },
    React.createElement('h2', null, 'تسجيل حساب جديد'),
    React.createElement('input', {
        type: 'text',
        placeholder: 'اسم المستخدم'
    }),
    React.createElement('input', {
        type: 'email',
        placeholder: 'البريد الإلكتروني'
    }),
    React.createElement('input', {
        type: 'password',
        placeholder: 'كلمة المرور'
    }),
    React.createElement(
        'button',
        { type: 'submit' },
        'تسجيل'
    )
);

Result | النتيجة

What's Happening? | ماذا يحدث؟

  1. createElement:

    Creates a React element with specified properties | ينشئ عنصر React بالخصائص المحددة

  2. createRoot:

    Establishes connection with DOM | يؤسس الاتصال مع DOM

  3. render:

    Displays the element in browser | يعرض العنصر في المتصفح
