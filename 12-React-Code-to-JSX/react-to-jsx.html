Lesson Twelve: React Code to JSX

الدرس الثاني عشر: تحويل كود React إلى JSX

في هذا الدرس سنتعلم كيفية تحويل كود React التقليدي إلى JSX مع مثال عملي.
In this lesson, we'll learn how to convert traditional React code to JSX with a practical example.

React.createElement | الطريقة التقليدية

الطريقة التقليدية باستخدام React.createElement:
Traditional way using React.createElement: 

// كود React التقليدي
const element = React.createElement(
    'div',
    null,
    React.createElement(
        'h1',
        null,
        'الوقت الحالي: ' + new Date().toLocaleTimeString()
    )
);

JSX Syntax | صيغة JSX

نفس الكود باستخدام JSX (أسهل في القراءة والكتابة):
Same code using JSX (easier to read and write): 

// نفس الكود باستخدام JSX
const element = (
    <div>
        <h1>
            الوقت الحالي: {new Date().toLocaleTimeString()}
        </h1>
    </div>
);

مثال عملي | Practical Example

لنقارن المثالين بشكل كامل:
Let's compare both examples in full: 

// React Traditional Code
const root = ReactDOM.createRoot(
    document.getElementById('root')
);

function tickWithReact() {
    const element = React.createElement(
        'div',
        null,
        React.createElement(
            'h1',
            null,
            'الوقت الحالي: ' + new Date().toLocaleTimeString()
        )
    );
    root.render(element);
}

setInterval(tickWithReact, 1000);

// JSX Version
const root = ReactDOM.createRoot(
    document.getElementById('root')
);

function tickWithJSX() {
    const element = (
        <div>
            <h1>
                الوقت الحالي: {new Date().toLocaleTimeString()}
            </h1>
        </div>
    );
    root.render(element);
}

setInterval(tickWithJSX, 1000);

Babel Compilation | ترجمة Babel

لاستخدام JSX، نحتاج إلى Babel لترجمة الكود:
To use JSX, we need Babel to compile the code: 

<!-- Add React and ReactDOM -->
<script src="https://unpkg.com/react@18/umd/react.development.js"></script>
<script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>

<!-- Add Babel for JSX support -->
<script src="https://unpkg.com/babel-standalone@6/babel.min.js"></script>

<!-- Your JSX Script -->
<script type="text/babel">
    // Your JSX code here
</script>

ملاحظة: يجب إضافة type="text/babel" للسكريبت الذي يحتوي على JSX.
Note: Add type="text/babel" to scripts containing JSX.
