Introduction | مقدمة

In this lesson, we'll learn how to write our first React code and understand how to convert pure JavaScript code to React. We'll use CDN links to get started quickly.

في هذا الدرس، سنتعلم كيفية كتابة أول كود React وفهم كيفية تحويل كود JavaScript النقي إلى React. سنستخدم روابط CDN للبدء بسرعة.

Pure JavaScript Example | مثال JavaScript النقي

Pure JavaScript Code | كود JavaScript النقي


// Pure JavaScript Implementation
const paragraph = document.createElement('p');
const text = document.createTextNode('My name is Mohammed Mostafa');
paragraph.appendChild(text);
document.getElementById('pure-js-output').appendChild(paragraph);

React Example | مثال React

React Setup | إعداد React



<script src="https://unpkg.com/react@18/umd/react.development.js" crossorigin></script>
<script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js" crossorigin></script>

React Code | كود React


// React Implementation
const p = React.createElement('p', null, 'My name is Mohammed Mostafa');
const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(p);

Understanding the Code | فهم الكود

Pure JavaScript Steps | خطوات JavaScript النقي

1

Create paragraph element | إنشاء عنصر الفقرة

const paragraph = document.createElement('p');
2

Create text node | إنشاء عقدة النص

const text = document.createTextNode('My name is Mohammed');
3

Append text to paragraph | إضافة النص إلى الفقرة

paragraph.appendChild(text);
4

Append to document | إضافة إلى المستند

document.body.appendChild(paragraph);

React Steps | خطوات React

1

Create element | إنشاء العنصر

const p = React.createElement('p', null, 'My name is Mohammed');
2

Create root | إنشاء العنصر الجذر

const root = ReactDOM.createRoot(document.getElementById('root'));
3

Render element | عرض العنصر

root.render(p);

Key Differences | الفروق الرئيسية

  • Code Length | طول الكود

    React requires less code for the same result | يتطلب React كودًا أقل لنفس النتيجة

  • Element Creation | إنشاء العناصر

    React combines creation and content in one step | يجمع React الإنشاء والمحتوى في خطوة واحدة

  • DOM Updates | تحديثات DOM

    React uses Virtual DOM for efficient updates | يستخدم React DOM افتراضي للتحديثات الفعالة
