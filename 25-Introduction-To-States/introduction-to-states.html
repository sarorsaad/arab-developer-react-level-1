Lesson Twenty-Five: Introduction to States

الدرس الخامس والعشرون: مقدمة في States

بسم الله الرحمن الرحيم

في هذا الدرس، سنتعرف على مفهوم الـ States في React وكيفية استخدامها لتخزين وإدارة البيانات في مكونات React.

In this lesson, we'll learn about React States and how to use them to store and manage data in React components.

States مقابل Variables | States vs Variables

الـ States في React تشبه المتغيرات (Variables) في الدوال العادية. كلاهما يستخدم لتخزين البيانات، لكن الـ States لها ميزات خاصة في React.

States in React are similar to variables in regular functions. Both are used to store data, but States have special features in React.

مثال على المتغيرات العادية | Example with Regular Variables:

function checkAge(age) {
    const ageLimit = 18; // متغير عادي | Regular variable
    
    if (age >= ageLimit) {
        return "Age is greater than " + ageLimit;
    }
    return "Age is less than " + ageLimit;
}

مثال عملي | Practical Example

لنرى مثالاً على استخدام الـ State في مكون React:

class UserProfile extends React.Component {
    state = {
        nameUpperCase: this.props.name.toUpperCase()
    }

    render() {
        return (
            <div>
                <h2>{this.props.name}</h2>
                <p>Your first letter is: {this.state.nameUpperCase.charAt(0)}</p>
            </div>
        );
    }
}

تعريف States | Declaring States

يتم تعريف الـ State كحقل عام (public field) في مكون الـ Class:

State is declared as a public field in the Class component:

class MyComponent extends React.Component {
    state = {
        // تعريف المتغيرات هنا | Define variables here
        count: 0,
        name: 'Ahmed',
        isActive: true
    }
}

استخدام States | Using States

للوصول إلى قيم الـ State، نستخدم this.state:

To access State values, we use this.state:

render() {
    return (
        <div>
            <p>Count: {this.state.count}</p>
            <p>Name: {this.state.name}</p>
            <p>Status: {this.state.isActive ? 'Active' : 'Inactive'}</p>
        </div>
    );
}

أفضل الممارسات | Best Practices

  • استخدم State لتخزين البيانات المتغيرة | Use State for data that changes
  • تجنب تكرار الكود باستخدام State | Avoid code duplication using State
  • اجعل أسماء State واضحة ومعبرة | Use clear and descriptive State names
  • لا تخزن بيانات يمكن حسابها من props أو state أخرى | Don't store data that can be computed from props or other state

الملخص | Summary

الـ States في React هي طريقة قوية لتخزين وإدارة البيانات في مكوناتك. فهي تشبه المتغيرات العادية ولكن مع ميزات إضافية خاصة بـ React.

States in React are a powerful way to store and manage data in your components. They are similar to regular variables but come with special React features.
