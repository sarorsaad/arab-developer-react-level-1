Lesson Twenty-Three: Type Checking Introduction

الدرس الثالث والعشرون: مقدمة في فحص الأنواع

بسم الله الرحمن الرحيم

في هذا الدرس سنتعرف على أهمية التحقق من أنواع البيانات (Type Checking) في React، وكيف يمكن أن يساعدنا في تجنب الأخطاء المنطقية.

In this lesson, we'll learn about the importance of Type Checking in React and how it helps us avoid logical errors.

مثال على المشكلة | Problem Example

لنرى مثالاً على مشكلة يمكن أن تحدث بدون Type Checking:

// مكون بسيط يعرض الاسم والعمر
class Greeting extends React.Component {
render() {
const ageAfterThreeYears = this.props.age + 3;
return (
<div>
Hello, your name is {this.props.name} and
your age after 3 years will be {ageAfterThreeYears}
</div>
);
}
}
// استخدام المكون مع قيم غير صحيحة
<Greeting name="Mohammed" age="20" /> {/* age as string! */}

الأخطاء المنطقية | Logical Errors

المشكلة في المثال السابق:

  • تم تمرير العمر كـ string بدلاً من number
  • عند إضافة 3 سنوات، سيتم دمج النصوص بدلاً من الجمع الحسابي
  • النتيجة ستكون "203" بدلاً من 23
  • لا يوجد تحذير أو خطأ يخبرنا بهذه المشكلة

حلول Type Checking | Type Checking Solutions

هناك حلان رئيسيان للتحقق من أنواع البيانات في React:

  1. PropTypes:
    • مكتبة مخصصة للتحقق من أنواع Props
    • سهلة الاستخدام ومناسبة للمشاريع الصغيرة
    • كانت مدمجة في React قبل الإصدار 15.5
  2. TypeScript:
    • لغة برمجة تضيف أنواع ثابتة إلى JavaScript
    • موصى بها من قبل فريق React
    • مناسبة للمشاريع الكبيرة

مكتبة PropTypes | PropTypes Library

مثال على استخدام PropTypes:

// تحديد أنواع Props باستخدام PropTypes
Greeting.propTypes = {
name: PropTypes.string.isRequired,
age: PropTypes.number.isRequired
};

TypeScript كحل بديل | TypeScript Alternative

مثال على نفس المكون باستخدام TypeScript:

// تعريف المكون باستخدام TypeScript
interface GreetingProps {
name: string;
age: number;
}
class Greeting extends React.Component<GreetingProps> {
// ... rest of the component
}

ملخص الدرس | Summary

تعلمنا في هذا الدرس:

  • أهمية التحقق من أنواع البيانات في React
  • كيف يمكن أن تؤدي الأخطاء في الأنواع إلى مشاكل منطقية
  • الحلول المتاحة للتحقق من الأنواع (PropTypes و TypeScript)
  • مقارنة بين الحلول المختلفة وتوصيات فريق React

في الدرس القادم، سنتعلم كيفية تثبيت واستخدام مكتبة PropTypes بالتفصيل.
